TheMoon

Кто такой TheMoon?

TheMoon - сильный инструмент для модерирования вашего Discord сервера


Команды делятся на несколько категорий

  • Модераторские команды, необходимые для поддержания порядка на вашем сервере
    • /ban - банит пользователя на сервере
    • /unban - разбанивает пользователя на сервере
    • /kick - выгоняет пользователя с сервера
    • /clear - очищает чат
    • /timeout - запрещает пользователю исполдьзовать чат на установленно количество времени
    • /create_role - создаёт роль на сервере
    • /delete_role - удаляет роль на сервере
    • /give_role - выдаёт роль пользователю
    • /auditlog - создаёт журнал аудита в установленном канале
  • Информационные команды, помогающие получить доступную информацию
    • /help - предоставляет пользователю полный список команд
    • /info - позволяет получить доступную информацию о пользователе
    • /serverinfo - предоставляет информацию о сервере
    • /avatar - даёт возможность скачать аватар пользователя
    • /currency - отправляет в чат стоимость Доллара/Евро в рублях
    • /stickers - позволяет скачать стикер существующий на сервере
  • Экономические команды, с помощью которых можно создать собственный рынок на вашем сервере
    • /award - выдаёт пользователю ежедневную награду в виде валюты Бота
    • /balance - позволяет ознакомиться с балансом пользователя
    • /shop - выводит список ролей, доступных для покупки
    • /add_shop - позволяет выставить на продажу в магазин роль пользователя
    • /buy - позволяет купить роль, выставленную в магазине
    • /remove_shop - позволяет убрать из магазина роль, выставленную вами на продажу
    • /pay - позволяет перевести деньги напрямую другому пользователю
  • Команды для работы с приватными голосовыми каналами
    • /vset - создаёт каналы для работы с приватными голосовыми каналами. Дальшнейшее управление реализуется с помощью кнопок
  • Команды для создания системы тикетов на вашем сервере
    • /tset - создаёт каналы для работы с тикетами. Дальшнейшее управление реализуется с помощью кнопок
